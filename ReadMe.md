# ReadMe

This repo contains all files used in my project **pulse** and **smart fan**

对各个文件夹的说明如下：

| folder           | folder function        |
| ---------------- | ---------------------- |
| **0-report**     | 两篇论文和作业说明     |
| **1-sim**        | 所有的原理图和仿真文件 |
| **2-experiment** | **smart fan** 最终视频 |
| **3-code**       | **smart fan** 使用代码 |
| **4-datasheet**  | 所使用芯片的 datasheet |
| **5-PCB**        | 最终绘制的PCB文件      |
